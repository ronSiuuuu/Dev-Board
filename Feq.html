Question-1: What are the different ways to select an element in the DOM?

  • getElementById("id") – Selects an element by its ID (returns one element).
  • getElementsByClassName("class") – Selects elements by class (returns multiple elements).
  • getElementsByTagName("tag") – Selects elements by tag name.
  • querySelector("selector") – Selects the first matching element using a CSS selector.
  • querySelectorAll("selector") – Selects all matching elements (returns a NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML : Retrieves or sets the HTML content inside an element.
    If the content includes HTML tags, they will be parsed and rendered.

  • innerText Retrieves or sets only the visible text inside an element.
    It ignores hidden elements and does not interpret HTML tags.
  • textContent -Retrieves or sets all text content, including hidden elements.
    It does not process HTML tags and displays them as plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to
a parent element instead of adding individual listeners to each child element.
The parent element listens for events from its children and determines which child was interacted with using the event.target property.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where an event starts from the target element (the element that was directly interacted with)
and then bubbles up to its parent elements in the DOM tree, triggering event listeners on each parent along the way.
